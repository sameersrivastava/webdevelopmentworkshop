HTML - Hypertext MarkUp Language

HTML is the MarkUp for a WebPage. It is how things are ordered and placed on a page. HTML is written using tags which represent elements.

Elements written with Tags

<html> ... </html>

Let's Write Some HTML

Save a file in a text editor as home.html

<!DOCTYPE html>
<html lang="en">
	Hello World!
</html>
